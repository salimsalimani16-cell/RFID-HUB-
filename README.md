# RFID Hub - Site Officiel

Bienvenue sur le dépôt officiel de **RFID Hub**, votre fournisseur de confiance de solutions RFID et NFC (13.56 MHz) en Algérie.

## À propos du projet
RFID Hub est une plateforme B2B dédiée à la fourniture de puces, cartes MIFARE, badges et accessoires RFID pour les secteurs de l'hôtellerie et de l'industrie. 

Notre mission est de simplifier l'approvisionnement local en éliminant les délais douaniers et en facilitant l'accès au matériel pour les 58 wilayas d'Algérie.

## Fonctionnalités
- **Catalogue dynamique :** Présentation des technologies MIFARE (Classic, Ultralight, DESFire, NTAG).
- **Tableau de compatibilité :** Guide rapide pour les professionnels (VingCard, Onity, SALTO, HID).
- **Formulaire de devis intelligent :** Système intégré permettant de suggérer la puce adaptée selon la serrure et de demander un échantillon.
- **Réactivité :** Système de notification par email via EmailJS pour une réponse rapide aux demandes clients.

## Technologie
- **Frontend :** HTML5, Tailwind CSS (pour une interface moderne et responsive).
- **Backend (Intégration) :** EmailJS pour la gestion des formulaires sans serveur complexe.
- **Hébergement :** GitHub Pages.
